How To Play

    General Rules for Sudoku

  • Sudoku grid consists of 9x9 spaces.
  • You can use only numbers from 1 to 9.
  • Each 3×3 block can only contain numbers from 1 to 9.
  • Each vertical column can only contain numbers from 1 to 9.
  • Each horizontal row can only contain numbers from 1 to 9.
  • Each number in the 3×3 block, vertical column or horizontal row can be used only once.
  • For more information on how to play Sudoku, click here.

    About this version of Sudoku

  • Try to complete the game with all tiles filed with the correct number, in the fastest time, and with as few errors as possible
  • When all tiles have a number present the game will end.
  • To enter a number onto the board, click the number you want from the right hand side and then click on where you would like to place it. You will know which number you have selected as it will have a GREENbackground
  • Click 'New Game' button to restart your current game
  • Click on the difficulty display button to change difficulty and start a new game
  • The notes button will allow you to enter any numbers you want into the board to aid you to work out the correct answer. To remove a note, enter the same number again or enter any number when not in notes mode. The buttons will turn BLUE when active.
  • The undo button and the redo button will respectively undo or redo the previous actions taken. This will not undo/redo any notes made.
  • Clicking on the timer will pause the game, clicking again will unpause. If the game is paused you will not be able to enter numbers into the board or use the undo/redo buttons
  • The number of errors you make will be tracked. For the perfect game, keep the counter at 0!
