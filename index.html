How To Play

General Rules for Sudoku

  • Sudoku grid consists of 9x9 spaces.
  • You can use only numbers from 1 to 9.
  • Each 3×3 block can only contain numbers from 1 to 9.
  • Each vertical column can only contain numbers from 1 to 9.
  • Each horizontal row can only contain numbers from 1 to 9.
  • Each number in the 3×3 block, vertical column or horizontal row can be used only once.
  • For more information on how to play Sudoku, click here.

About this version of Sudoku

  • The goal of the game is to fill in all the tiles with the correct numbers in the shortest amount of time and with as few mistakes as possible.
  • The game will end when all tiles have been filled in.
  • To place a number on the board, first click your desired number from the list on the right handside. When selected it will have a GREEN background. Then click on the tile you wish to change on the board.
  • The 'Notes' button allows you to enter any numbers to aid you in finding the correct answer. When active, the notes button will have a BLUE background. To remove a note, enter the same number again or enter any number when not in notes mode (be sure you enter the right number to avoid an error!)
  • To restart the current game, click the 'New Game' button.
  • To change the difficulty and start a new game, click on the difficulty display button.
  • The 'Undo' and 'Redo' buttons allow you to undo or redo previous actions, but will not undo or redo any notes made.
  • Clicking on the timer will pause the game, clicking again will unpause. When the game is paused, you will not be able to enter numbers or use the undo/redo buttons.
  • The number of mistakes you make will be tracked, aim for a perfect game by keeping the counter at 0
